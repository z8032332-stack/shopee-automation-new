```
蝦皮聯盟後台選品
      ↓

```

## 第二步：批量生成文案

使用 Claude Artifact 工具（shopee excel generator）自動生成。

### 注意事項
- 螢幕逾時設定改為最長（或開啟「檢視時保持螢幕開啟」）
- 不可切換對話、不可關閉頁面（結果存在瀏覽器記憶體）
- 每跑 20 筆按一次「📊 匯出 Excel」備份
- 預估時間：100 筆約 8-12 分鐘

### 口播文案結構
```
第1-2行：開場鉤子（0-5秒）
第3-4行：主題說明（6-10秒）
第5-6行：問題鋪陳（11-15秒）
第7-8行：解決辦法（16-20秒）
第9行：成果想像（21-25秒）
第10行：購買 CTA（結尾）
```

### CTA 規則
- 連結位置說「下方」，不說「簡介」
- 每筆 CTA 不同，活潑有創意
- 範例：還不快加入購物車。／慢了就沒有喔。／今天不買明天漲。／不買會後悔的啦。

## 第三步：合規檢查（每筆文案必須套用）

### 禁止推廣品類
- 藥品（藥字號）、菸酒、電子菸
- 活體動植物、情趣用品
- 棉花棒、化妝棉等醫療耗材

### 化妝品／保養品
❌ 消炎、殺菌、除疤、治療、保證美白、永久美白
✅ 保濕、滋潤、提亮膚色、修飾毛孔外觀

### 食品／保健品
❌ 減肥保證、降血壓、治失眠、抗癌、保證瘦
✅ 幫助消化、調整體質、促進新陳代謝、維持體態

### 健康器材
❌ 治好、治癒、根治、保證復原
✅ 支撐保護、穿戴舒適、使用後更有支撐感

### 通用禁詞
❌ 100%有效、保證見效、全球第一、立即見效、最強、永久
✅ 多人好評、持續使用更有感、使用後感覺明顯

## 第四步：清理空白關鍵字商品

執行以下 Python 程式（或請 Claude 執行）：

```python
import openpyxl

wb = openpyxl.load_workbook("蝦皮分潤前100_含文案.xlsx")
ws = wb.active

to_delete = []
for i, row in enumerate(ws.iter_rows(min_row=2, values_only=False), start=2):
    if not row[6].value or str(row[6].value).strip() == "":
        to_delete.append(i)

for idx in reversed(to_delete):
    ws.delete_rows(idx)

# 重新編號
for i, row in enumerate(ws.iter_rows(min_row=2), start=1):
    row[0].value = i

wb.save("蝦皮分潤前100_整理版.xlsx")
print(f"完成，剩餘 {ws.max_row - 1} 筆")
```

## 常見問題

**Q：文案生成失敗怎麼辦？**
按「🔄 重跑失敗筆」按鈕，程式會自動重試失敗的商品。

**Q：跑到一半被中斷？**
重新開啟工具，按「▶ 繼續」會從中斷處接著跑（前提是同一個對話未關閉）。

**Q：切換分頁會暫停嗎？**
會。保持分頁在前景，或設定螢幕逾時為最長避免暗屏中斷。

**Q：Token 費用？**
100 筆文案約 NT$15-30，使用 Claude.ai 訂閱方案不額外收費。
