# Shopee 關鍵字選品 Skill B
## 每月更新版｜當前版本：2026年3-4月

---

## 一、Skill 說明

這個 skill 用於根據「當月＋下月蝦皮熱門關鍵字」動態選品，每月20號後由使用者提供新關鍵字更新。

與 Skill A（熱銷前50排行）完全獨立，兩者可交叉使用，同一個月內輪流跑。

---

## 二、關鍵字更新規則

- **更新時機**：每月20號後，蝦皮公布當月關鍵字與下月預測後
- **更新方式**：使用者提供關鍵字清單 → Claude 更新下方「當前關鍵字區塊」→ 存回 skill 檔案
- **若超過20號使用者尚未更新**：Claude 應主動提醒：「本月關鍵字尚未更新，請提供最新蝦皮關鍵字」

## 四、搜尋與抓取邏輯

### 搜尋順序
1. 先跑當月**亮點關鍵字**（權重最高，爆款機會）
2. 再跑當月**一般關鍵字**
3. 最後補下月預測關鍵字（填滿50筆用）
4. 合併後去重，確保不重複商品

### 搜尋參數
- **每個關鍵字抓取數量**：前20筆（按銷量排序）
- **最低銷量門檻**：500以上（確保是消費者真實在買的商品）
- **隨機因子**：同一關鍵字符合條件的商品中，隨機抽取，不固定取銷量第1名，確保每次跑出來的結果不同
- **最終輸出**：50筆，不足則放寬到銷量300

### 過濾條件
- 必須有蝦皮聯盟分潤連結
- 排除違禁品類：藥品、酒類、菸草、棉花棒、化妝棉
- 排除已在本月 Skill A 熱銷清單中出現的商品（避免重複）

## 五、輸出格式

輸出為 Excel，欄位如下：

| 欄位 | 說明 |
|------|------|
| A - 品名 | 商品標題 |
| B - 分潤連結 | 蝦皮聯盟連結 |
| C - 價格 | 商品售價 |
| D - 分潤率 | 佣金比例 |
| E - 銷量 | 已售數量 |
| F - 對應關鍵字 | 此商品從哪個關鍵字搜到的 |
| G - 口播文案 | 留空，後續跑文案 skill 填入 |
| H - 關鍵字文案 | 留空，後續跑文案 skill 填入 |
| I - 狀態 | 留空 |

## 六、每月更新 SOP

```
步驟1：使用者說「更新關鍵字」或「20號到了」
步驟2：Claude 詢問：「請提供本月關鍵字（各品類＋亮點）和下月預測關鍵字」
步驟3：使用者貼上蝦皮公布的關鍵字清單
步驟4：Claude 只更新第三節「當前關鍵字區塊」，其餘邏輯不動
步驟5：存回 /mnt/skills/user/shopee_keyword_selection_skill.md
步驟6：通知使用者「關鍵字已更新，可以跑選品了」
```
